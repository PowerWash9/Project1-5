About the Course:

CIS 164 - JavaScript

This course covers client-side Web scripting using JavaScript along with other Web page development tools. It covers many programming techniques of JavaScript, and so far we have learned about creating scripts in HTML files, and creating separate script files to include in the HTML files. Some programs we have created so far include converting temperature (Celsius and Fahrenheit), making change (determining number of quarters, pennies, etc.), determining sales tax, using arrays to determine totals in certain regions, and creating an income tax calculator found below.
